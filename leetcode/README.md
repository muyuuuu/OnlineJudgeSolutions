# 题解整理

注：这里的题解会很粗糙。一些约定：
- TO，timeout，表示超时
- WA，wrong answer，表示错误

## 题目类型

1. advanced-DS，高级数据结构，如 LRU，LFU
2. array，数组相关
3. binary search，二分查找
4. classical-algo，经典算法
5. dfs bfs，回溯与遍历
6. divide-conquer，分治算法
7. dynamic-programming，动态规划
8. hash，哈希算法
9. heap，优先级队列
10. intervals，区间问题
11. link，链表
12. math，数学运算
13. slidewindow，滑动窗口
14. stack，栈
15. tree，树
16. unionfind，并查集
